# banglejs2-sphclock

Hacky clock for my bangle.js 2, with a "vintage menu kind of 50s" look.

## Features

- CLOCK!
- Next appointments in schedule (from json)
- Battery level
- Date, day of week
- Current and future weather as a graph (from json)
- Click-to-open-calendar
